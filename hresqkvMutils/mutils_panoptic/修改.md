# dual_diff_refine_unet.py 替换 Unet.py 的修改清单

1. 替换导入（mutils_panoptic/MuTILs.py）
   - 将 `from mutils_panoptic.Unet import UNet` 改为 `from mutils_panoptic.dual_diff_refine_unet import DualDiffRefineUNet`。

2. 更新配置参数名（configs/panoptic_model_configs.py 及 results/.../panoptic_model_configs.py）
   - `in_channels` 改为 `in_channels_left` / `in_channels_right`。
   - 删除 `external_concat_layer` / `external_concat_nc`（DualDiffRefineUNet 不支持）。
   - 新增可选参数：`tap_depth`、`inject_layer`、`refine_channels`、`diff_mode`、`attn_d_model`、`attn_heads`、`attn_dropout`、`fuse`。
   - 示例（需与实际深度/通道一致）：
     - `roi_unet_params`: `{'in_channels_left': 3, 'in_channels_right': 3, 'depth': 5, ...}`
     - `hpf_unet_params`: `{'in_channels_left': 3, 'in_channels_right': 3, 'depth': 5, ...}`

3. 替换实例化（mutils_panoptic/MuTILs.py）
   - 将 `self.roi_unet = UNet(**self.roi_unet_params)` 改为 `self.roi_unet = DualDiffRefineUNet(**self.roi_unet_params)`。
   - 将 `self.hpf_unet = UNet(**self.hpf_unet_params)` 改为 `self.hpf_unet = DualDiffRefineUNet(**self.hpf_unet_params)`。

4. 处理 “双输入” 要求（mutils_panoptic/MuTILs.py）
   - DualDiffRefineUNet 需要 `x_left` 与 `x_right`，且空间尺寸必须一致。
   - 若没有天然双输入，可暂时用同一张图（或轻微增强）：
     - `roi_logits = self.roi_unet(lowres_rgb, lowres_rgb)`
     - `hpf_nuclei_pre = self.hpf_unet(hpf_rgbs, hpf_rgbs)`

5. 处理 ROI 中间特征 fetch（关键差异）
   - 原逻辑：`roi_logits, roi_intermed = self.roi_unet(..., fetch_layers=[self.rilayer])`
   - DualDiffRefineUNet 不支持 `fetch_layers`，因此必须二选一：
     A) 改 DualDiffRefineUNet：增加 `return_tap` / `return_features` 之类的返回值，再在 MuTILs 里使用返回的中间特征来替代 `roi_intermed`。
     B) 移除 ROI->HPF 的中间特征通路：删掉 `roi_interm_layer` / `hpf_interm_layer` / `roi_intermed` 相关计算与 `roi_align`，并同步删除 `hpf_unet` 的 `cx` 依赖。

6. 修改 HPF 前向调用
   - 原逻辑：`hpf_nuclei_pre = self.hpf_unet(hpf_rgbs, cx=hpf_roi_intermed)`
   - 新逻辑：`hpf_nuclei_pre = self.hpf_unet(hpf_left, hpf_right)`（确保两路尺寸一致）。
   - 若你仍想融合 `roi_intermed`，需要额外做：对 `roi_intermed` 进行通道投影 + 尺寸对齐，再作为 `x_right` 输入（或修改 DualDiffRefineUNet 支持外部注入）。

7. 兼容性与权重加载
   - DualDiffRefineUNet 的参数命名与层结构都不同，旧 UNet 权重不可直接加载。
   - 若需迁移旧权重，需手工映射 key 或使用 `strict=False` 并接受部分未加载。
